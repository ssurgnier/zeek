# zeek [![Build Status](https://secure.travis-ci.org/ssurgnier/zeek.png?branch=master)](http://travis-ci.org/ssurgnier/zeek)

A simple zookeeper cli.

## Usage
```
lein run <path/to/config> <env>
/ $ ls
(znode1 znode2 znode3)
/ $ cd znode1
/znode1 $
```
